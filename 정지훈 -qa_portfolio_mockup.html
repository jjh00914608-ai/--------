Growth Roadmap

현재 포트폴리오는 Manual QA 기반입니다. 이후 API 검증, 자동화, 배포 파이프라인 이해로 확장 가능한 구조로 목표를 잡았습니다.

STEP 01

Manual QA 기본기

요구사항 분석, 테스트케이스 설계, 재현 가능한 버그 리포트 작성 완성도 향상

STEP 02

API Testing

Automation

CI/CD 협업
